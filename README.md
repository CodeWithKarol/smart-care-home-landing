# SmartCare Home Landing Page

A modern, high-converting landing page template designed for smart home care monitoring solutions. This template features a premium aesthetic, responsive design, and smooth scroll animations.

## 🚀 Key Features

*   **Modern & Premium UI**: Clean layout with glassmorphism effects, gradients, and pill-shaped buttons.
*   **Fully Responsive**: Optimized for all devices, from large desktops to mobile phones.
*   **Scroll Animations**: Elegant entrance animations for all major sections using Intersection Observer.
*   **Interactive Elements**:
    *   Floating hero cards
    *   Hover effects on feature and testimonial cards
    *   Mobile-friendly navigation menu
*   **Redesigned Sections**:
    *   **Hero**: High-impact visual with trust indicators and adjusted mobile padding.
    *   **Get Started**: Clear pricing/action cards with distinct visual hierarchy.
    *   **Footer**: Comprehensive 4-column layout with a pre-footer CTA, dark theme, and centered mobile layout.
*   **Updated Branding**: Includes the latest FontAwesome 6.5.1 icons (including the X logo).

## 🛠️ Technologies Used

*   **HTML5**: Semantic markup for better accessibility and SEO.
*   **CSS3**:
    *   CSS Grid & Flexbox for layout.
    *   CSS Variables for easy theming.
    *   Keyframe animations and transitions.
*   **JavaScript (Vanilla)**:
    *   `IntersectionObserver` for scroll animations.
    *   Mobile menu toggling.
    *   Smooth scrolling navigation.
*   **FontAwesome 6.5.1**: For modern, scalable icons.

## 🔧 Setup & Customization

1.  **Clone or Download**: Download the project files to your local machine.
2.  **Open**: Open `index.html` in any modern web browser to view the page.
3.  **Edit**:
    *   Modify `index.html` to update text, links, and images.
    *   Adjust colors and fonts in the `:root` section of `styles.css`.
    *   Replace image URLs with your own assets (currently using Unsplash placeholders).
