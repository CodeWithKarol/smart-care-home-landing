Step 1

Branding & Colors

The template uses CSS variables for easy color management. You can change the entire color scheme by editing just a few lines in styles.css.

Locate the Variables

Open styles.css and look for the :root selector at the very top of the file.

:root {
    --primary-color: #0066cc;   /* Main brand color (Buttons, Links) */
    --secondary-color: #00b894; /* Accent color (Gradients, Success states) */
    --accent-color: #f39c12;    /* Highlight color (Stars, Warnings) */
    --dark-text: #2c3e50;       /* Headings */
    --light-text: #7f8c8d;      /* Body text */
}
Tip: The "Get Started" and "Footer" sections use gradients derived from the primary and secondary colors. Changing these two variables will automatically update those gradients.
Step 2

Editing Content

All text content is located in index.html. The file is structured with clear comments marking each section.

Key Sections to Update:

  • Navigation: Update the logo text "SmartCare" inside the .nav-logo class.
  • Hero Section: Change the <h1> headline and the subheadline paragraph.
  • Contact Info: Update the phone number and email in the Footer section (approx. line 1150).
  • Legal: Don't forget to update the copyright year and company name in the footer bottom.
<!-- Example: Updating the Hero Title -->
<h1 class="hero-title">
    Your New Headline Here <br />
    <span class="text-gradient">Highlighted Text</span>
</h1>
Step 3

Updating Images

The template currently uses Unsplash placeholder images. You should replace these with your own high-quality photography.

Image Locations:

  • Hero Image: Look for .hero-main-image img.
  • Feature Images: Inside the .benefit-row divs.
  • Testimonial Avatars: Inside .testimonial-author img.
  • Mobile Menu Image: Inside .mobile-menu-image img.
<img 
    src="path/to/your-image.jpg" 
    alt="Descriptive text for accessibility" 
/>
Important: For the best performance, optimize your images before uploading. Use WebP or compressed JPG formats.
Step 4

Changing Icons

This template uses FontAwesome 6.5.1. You can choose from thousands of free icons.

  1. Go to the FontAwesome Gallery.
  2. Search for an icon (e.g., "heart", "shield", "user").
  3. Copy the HTML class (e.g., fa-solid fa-heart).
  4. Replace the <i> tag class in your HTML.
<!-- Changing the Twitter logo to X -->
<a href="#" aria-label="X">
    <i class="fa-brands fa-x-twitter"></i>
</a>
Step 5

Configuring Forms

The template includes modal forms for "Schedule Demo", "Learn More", and "Contact". Currently, these forms prevent default submission and log the data to the console.

Connecting to a Backend

To make the forms functional, you need to edit the handleFormSubmit function in script.js or change the form's action attribute in HTML.

Option A: Use a Service (Easier)
Use a service like Formspree or Netlify Forms. Just add the action URL to the form tag:

<form action="https://formspree.io/f/your-id" method="POST">

Option B: Custom JavaScript
Edit script.js to send an AJAX/Fetch request to your API.

Step 6

Animations

The page features scroll-triggered entrance animations. These are handled by the IntersectionObserver in script.js.

How it works:

  1. Elements to be animated are defined in the elementsToObserve list in script.js.
  2. CSS hides them initially (opacity: 0).
  3. When they scroll into view, JS adds the class .animate-in.
  4. CSS plays the fadeUp animation.

If you add a new section and want it to animate, simply add its class to the selector list in script.js:

const elementsToObserve = document.querySelectorAll(
    ".process-step, ..., .your-new-class"
);
